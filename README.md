# Benefique Financial Times (BFT)

A financial dashboard and reporting platform for fractional CFO services and real-time accounting insights.

## Overview

Benefique Financial Times is a web-based financial intelligence platform built for CPAs, EAs, and accounting firms to deliver real-time financial insights to their clients.

## Features

- **Real-time Books** - Live financial position updates
- **Monthly Closes** - Automated close workflows
- **Cash Flow Forecasting** - Project future cash positions
- **Dashboard Analytics** - Visual financial metrics
- **Client Reports** - Professional financial summaries

## Tech Stack

- React/Vite - Modern frontend architecture
- Node.js/Express - Backend services
- PostgreSQL - Data persistence
- Recharts - Financial visualizations

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/gerrit-design/benefique-financial-times.git
cd benefique-financial-times
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
